# Hello Astro.js

## 为什么？

1. 内容驱动。互联网的初衷。
2. UX 体验非常好，简化了许多配置工作。
3. Hydration 平衡 SSR 与 SPA。
4. 可以任意使用 react.js 等 UI 框架。
5. 支持 API，配合 htmx 或 alpine.js 这样的框架也非常不错。
6. `.astro` 组件，近似于原生的体验，同时分离前后端 JS 代码。
7. 性能优化，静态资源，懒加载。

## 如何开始使用？

[官方文档](https://docs.astro.build/en/getting-started)！完成 `Tutorial: Build a Blog` (0~2h) 后根据需要查阅其他文档即可。

## 感受

没有那么多「约定」和让人「困惑」的行为。（~next.js: ???~）

~绝佳的 WebDev Playground 可以尝试体验许多库/框架，无需额外搭建环境。~
